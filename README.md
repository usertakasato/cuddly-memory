# Cuddly Memory

Basit, sevimli temalı bir Memory (eşleştirme) oyunu — HTML/CSS/JS ile yazıldı. 4x4 (8 çift) kart içerir.

Nasıl çalıştırılır
1. Bu dosyaları aynı klasöre koyun: `index.html`, `styles.css`, `script.js`, `README.md`.
2. Tarayıcınızda `index.html` dosyasını açın (çift tıklayın veya tarayıcıya sürükleyin).
3. Kartlara tıklayarak eşleştirme yapın. Tüm kartları eşleştirdiğinizde süreniz ve hamle sayınız gösterilir.
4. "Yeniden Başlat" butonuyla oyunu sıfırlayabilirsiniz.

Özellikler
- Karıştırma (shuffle)
- Hamle sayacı
- Eşleşme sayacı
- Sayaç (timer)
- Basit flip animasyonu ve mobil uyum

Geliştirmek isterseniz (öneriler)
- Daha fazla seviye (6x6, 5x4 vb.)
- Skor tablosu (localStorage ile)
- Görsel kartlar yerine dışarıdan resim yükleme
- Animasyon/puanlama sistemi
